# FarReach
In FarReach, explore deep space, scavenge materials, and collect upgrades to survive longer each run. Death is inevitable, but progress carries forward. Push past your limits and see how far you can go before the void claims you.

Playable browser prototype.

Controls:
- WASD / Arrow Keys – Move
- Shift – Boost
- Space – Brake
- E – Interact
- Mouse Wheel – Zoom
- N – New Game

Built as a fast MVP.
